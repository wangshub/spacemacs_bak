# Spacemacs 配置文件备份

## 安装emacs25
```
sudo add-apt-repository ppa:kelleyk/emacs
sudo apt-get update
sudo apt-get install emacs25

```

## 安装spacemacs
```
git clone -b develop https://github.com/syl20bnr/spacemacs ~/.emacs.d
```

## 恢复配置
```
git clone https://github.com/wangshub/spacemacs_bak.git ~/.spacemacs.d
```
